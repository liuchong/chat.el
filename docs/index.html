AI Chat for Emacs

A pure Emacs AI chat client focused on coding workflows. Multi-turn conversations, tool calling, file operations, and session persistence — all without leaving your editor.

Built for Coders

Everything you need to supercharge your Emacs workflow with AI assistance

Multi-Turn Chat

Natural conversations with context preservation and intelligent trimming. Your chat history stays organized and accessible.

🔌

Multiple Providers

Connect to Kimi, Kimi Code, and any OpenAI-compatible API. Switch providers seamlessly without changing your workflow.

🛠️

Tool Calling

Let AI read files, search code, and execute commands. Built-in approval gates keep risky operations under your control.

File Operations

AI-assisted file reading, grep, and modification with configurable safety controls. Works within allowed directories only.

Session Management

Streaming Display

Real-time response streaming through async non-blocking UI. Watch the AI think and respond in real-time.

Quick Start

Get up and running in under a minute

emacs-lisp 
;; 1. Add to your load path
(add-to-list 'load-path "/path/to/chat.el")
(require 'chat)

;; 2. Configure your provider
(setq chat-default-model 'kimi-code)
(setq chat-llm-kimi-code-api-key "your-api-key")

;; 3. Or use auth-source for secure storage
;; machine kimi-code-api user api-key password YOUR_KEY

;; 4. Start chatting
;; M-x chat

Modular Architecture

Runtime modules are organized under `lisp/core`, `lisp/llm`, `lisp/tools`, `lisp/ui`, and `lisp/code`

chat.el

Entry point and command wiring

lisp/ui/chat-ui.el

Chat buffer rendering and response lifecycle

lisp/core/chat-session.el

Session and message persistence

lisp/llm/chat-llm.el

Provider abstraction and async request handling

lisp/core/chat-stream.el

SSE parsing and chunk handling

lisp/tools/chat-tool-caller.el

Tool prompt contract, parsing, and execution

lisp/core/chat-approval.el

Approval flow for risky tools and tool creation

lisp/core/chat-files.el

Built-in file tools and path safety checks

lisp/core/chat-context.el

Context trimming and summary generation

lisp/tools/chat-tool-forge.el

Tool registry, compilation, loading, and execution

lisp/code/chat-code.el

Code mode entry point and programming workflow

lisp/tools/chat-tool-forge-ai.el

AI-assisted tool generation flow
